# S-35: Frontmatter Parser

## Why
All CLI state reading requires parsing flat YAML frontmatter from .greenlight/slices/*.md files. This is the foundation package.

## What
Parse and write flat key-value YAML frontmatter. Simple line-by-line string splitting (D-43, stdlib only).

## Dependencies
None (wave 23, independent)

## Contracts
- C-91: FrontmatterParse
- C-92: FrontmatterWrite

## Decisions
- Line-by-line string splitting (D-43) — no YAML library needed
- Split on first colon only — preserves URLs and timestamps
- Whitespace-only lines skipped between delimiters
- Keys written in alphabetical order for deterministic output
