# S-42: Single Slice Command

## Why
Users need to run a single slice headlessly from the terminal, or auto-detect the first ready slice when no ID is provided.

## What
RunSlice in internal/cmd/slice.go handles single slice execution: parse args for slice ID and flags, detect context (shell vs Claude), spawn headless Claude in shell mode, output info in Claude mode. Auto-detect picks first ready slice by wave/ID order when no ID provided.
